Présentation de l’application

ChartDraw est une petite application Web pour manipulation de données. Elle a pour but d’effectuer des calculs statistiques et d’afficher les graphes correspondants. Elle aide ainsi les data analystes à mieux interpréter les données. Son avantage est la possibilité de traiter une large dataset en ligne sans devoir installer aucun logiciel.

Donc c'est mieux d'utiliser ChartDraw. Il est en version alpha pour le moment. Et n'hesiter pas a nous envoyer vos commentaire ou remarque.
